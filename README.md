This **Dice Simulator** is a Python program that uses the NumPy library to simulate rolling a dice either once or multiple times. It provides an interactive menu where the user can choose to roll a single dice or perform multiple rolls at once. When rolling multiple times, the program generates random values between 1 and 6, stores the results, and displays both the outcomes and the frequency of each number.This program demonstrates concepts such as functions, loops, user input handling, random number generation, and basic data analysis using NumPy.
